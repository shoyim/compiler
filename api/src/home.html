Ko'p sonli dasturlash tillarini qo'llab-quvvatlovchi online judge tizimi. Kodingizni yuboring, natijani real vaqtda oling.

Tizimga kirish API hujjatlari

Real vaqtda bajarish

Kod yuborilishi bilanoq izolyatsiyalangan muhitda ishga tushiriladi va natija soniyalar ichida qaytariladi.

Xavfsiz sandbox

Har bir bajarish isolate sandboxi orqali to'liq izolyatsiyada amalga oshiriladi. CPU va xotira cheklovlari qo'llaniladi.

Ko'p tilli qo'llab-quvvatlash

Python, C++, Java, Rust, Go va boshqa o'nlab tillarda ishlaydi. Har bir til uchun bir necha versiya mavjud.

Avtomatik tekshirish

AC, WA, PE, TL, RE, CE — barcha judge verdiktlarini qo'llab-quvvatlaydi. Maxsus checker ham yozish mumkin.

stdin / stdout

Standart kiritish-chiqarish hamda fayl orqali ishlash ikkalasi ham to'liq qo'llab-quvvatlanadi.

REST API

To'liq hujjatlashtirilgan REST API orqali istalgan platformaga integratsiya qilish mumkin. Token asosida autentifikatsiya.
