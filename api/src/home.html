Barcha masalalarni standart kiritish-chiqarish oqimlari orqali yechish mumkin (klaviatura orqali kiritish hamda ekranga chiqarish). Demo: maks 5 soniya, 64 MB.

Matnli oqim

Standart stdin/stdout orqali ishlaydi

print(sum(map(int, input().split())))

Faylli oqim

input.txt / output.txt orqali ishlaydi

fin = open("input.txt", "r")
fout = open("output.txt", "w")
a, b = map(int, fin.readline().split())
fout.write(str(a + b) + "\n")
fin.close()
fout.close()
